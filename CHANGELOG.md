# Changelog

## [1.0.0] - 2025-04-19

### 🌐 The Path Begins: BetterURL Awakens

- First version of `BetterURL`.
- Support for:
  - Authentication (`auth`).
  - Port (`portNumber`)
  - Paths (`path`)
  - Query parameters (`query`, `setQuery`, `tryQuery`, `removeQuery`, `clearQuery`)
  - Fragment (`fragment`)
- Fluent API based on the JavaScript `URL` object.
